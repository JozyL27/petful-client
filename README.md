## Summary
- Welcome to the Petful pet shop. You may adopt a pet by pressing the adopt button below. Petful uses a queue, so add your name to the list and wait for your turn to adopt a pet.

# Tech used
- Back-end:
    - Server: Node.js

    - server framework: Express.js

- Front-end:

    - Framework: React.js

    - HTML5, CSS3

- deployment:

    - Heroku & Vercel
